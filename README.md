# perf

Matrix multiplication performance benchmarks of various popular programming languages.

# Languages

* Java (Adoptium Temurin 18)
* C (latest GCC, includes -O2 and -O3 optimizations)
* C++ (latest G++, includes -O2 and -O3 optimizations)
* Python (latest)
* Go (1.18.4)
* Rust (latest stable)
